# GoldMaker

GoldMaker is a Lua addon that automates item selling and allows users to configure their own sellable item database.

The addon integrates with item hyperlinks and allows players to easily register items for automated selling.

---

## Features

- Automatic item selling
- Hyperlink integration for item configuration
- Persistent item database
- User configurable system
- Lightweight and extensible design

---

## How It Works

Players can add items to the selling list directly from the item hyperlink menu.

The addon stores this information in a persistent database so the configuration is preserved between sessions.

---

## Technologies

- Lua
- Event Driven Systems
- Persistent Data Storage
- UI Integration
